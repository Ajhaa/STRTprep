* STRTprep

STRTprep is an open-source software package for preprocess of STRT
RNA-seq data and for the analysis. The latest software codes are
maintained on GitHub (https://github.com/shka/STRTprep), and the
document is in the Wiki (https://github.com/shka/STRTprep/wiki).
